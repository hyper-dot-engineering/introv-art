# introv.art

**wear your quiet** — AI-generated introvert-themed apparel.

## About

introv.art is a small-batch apparel brand for introverts. Every design is AI-generated and curated with care. Print-on-demand, no waste.

## Tech Stack

- Static HTML/CSS (no frameworks)
- GitHub Pages hosting
- AI-generated artwork (OpenAI DALL-E / GPT Image)

## Development

Just open `index.html` in a browser. No build step needed.
